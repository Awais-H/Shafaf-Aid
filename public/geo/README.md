# GeoJSON Data

This folder should contain TopoJSON or GeoJSON files for the map visualization.
Each file should be named `{country_id}.geojson`.

## Required Files

- `syria.geojson`
- `yemen.geojson`
- `sudan.geojson`
- `afghanistan.geojson`
- `somalia.geojson`
- `palestine.geojson`
- `myanmar.geojson`
- `ethiopia.geojson`

## Data Sources (Recommendations)

- [HighSoft Map Collection](https://code.highcharts.com/mapdata/)
- [Humanitarian Data Exchange](https://data.humdata.org/)
- [GADM](https://gadm.org/)

## Placeholder Strategy (For Hackathon)
If real GeoJSONs are invalid or too large, use simplified bounding boxes or simplified polygons.
